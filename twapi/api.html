TWAPI.js Reference


TWAPI.js

TWAPI.js is a purely javascript framework for accessing Twitch chat and most of the Twitch API. It was created for use in KoalaBot mostly, but it has many other useful applications.

Why should you use TWAPI.js when TMI.js exists? No idea. Use either or both or neither.

Its intended use is for client-side web apps, CLI/BrowserSource pages for OBS, and NW.js/Electron desktop apps. It will not work with vanilla node.js due to the use of document.(event funtions) and browser-based websockets.

To use it, get the latest version, then create a minified version with closure-compiler.

An example test.html shows how to use TWAPi.js.

Listener - twapiRaw

Every RAW TMI message. You most likely won't be using this unless you need to parse for something that TWAPI.js doesn't already have a listener for.

Callback

  • .detail (string) - The raw command..

Example

  • document.addEventListener('twapiRaw', function( e ) { console.log(e.detail); } );
